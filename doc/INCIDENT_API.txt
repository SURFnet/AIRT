AIR_getIncidentById($id)
    
    $id: id of the requested incident
    Returns: AIR_Incident
    
    Get incident by id. If the incident does not exist, the returned incident
    will have an id of -1.

AIR_addIncident($incident)
    
    $incident: instance of AIR_Incident class representing the new incident
    Returns: nothing

    Add the given incident

AIR_updateIncident($incident)

    $incident: instance of AIR_Incident class representing the updated incident
    Returns: nothing

    Update the given incident

AIR_getIncidents()

    Returns: an multi dimensional array representing all incidents

    Return all incidents

AIR_associate($incident, $ticket)
    
    $incident: incident to associate
    $ticket: ticket to be associated

    Associate $incident with ticket $ticket.
